class Poly\+Mesh2\+D\+::\+Functional\+::\+Function$<$ input\+\_\+dim, output\+\_\+dim $>$}

The \mbox{\hyperlink{classPolyMesh2D_1_1Functional_1_1Function}{Function}} class is a wrapper class which stores a std\+::function describing the function value and another std\+::function describing the function derivative. ~\newline
~\newline
It describes in an abstract sense a function that maps elements from $\mathbb{R}^n \to \mathbb{R}^m$, for arbitrary postive integers $n, m$. ~\newline
~\newline
It is the base class for each of Scalar\+Function1D, Scalar\+Function2D, Vector\+Function1D, Vector\+Function2D, as well as \mbox{\hyperlink{classPolyMesh2D_1_1Functional_1_1Curve}{Curve}}. It is the core object in the \mbox{\hyperlink{namespacePolyMesh2D_1_1Functional}{Functional}} module.
