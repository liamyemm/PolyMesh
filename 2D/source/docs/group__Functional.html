Given two functions \(g:\mathbb{R}^n\to\mathbb{R}^k\) and \(f:\mathbb{R}^k\to\mathbb{R}^m\), compose constructs a Function <n, m> for the composition \(f\circ g:\mathbb{R}^n\to\mathbb{R}^m\).

Template Parameters
input_dim( \(n\) in description above) - The input dimension of the first function, \(g\). Must be a strictly positive integer value.
intermediate_dim( \(k\) in description above) - The output dimension of the first function, \(g\), and input dimension of the second function, \(f\). Must be a strictly positive integer value.
output_dim( \(m\) in description above) - The output dimension of the second function, \(f\). Must be a strictly positive integer value.

const ScalarBasis1D PolyMesh2D::Functional::MonomialScalarBasis ( const Curve &  param,
const size_t  degree 
)

Generates a scalar-valued shifted monomial basis in the parameter of a Curve. i.e. If the Curve is defined by \(E=\{\gamma(t)\}\) then the basis is in the coordinate \(t\).

Parameters
paramThe parameterisation of the Curve \(E\). Required to be affine (i.e. a straight line segment) in order for the space to be equivalent to the restriction of the polynomial space \(\mathbb{P}^k(\mathbb{R}^2)\) to the Curve \(E\).
degreeThe degree of the polynomial space.

Generates a scalar-valued monomial basis in 2 dimensions.

Parameters
transformdescribes a coordinate transformation under which the monomial is described. i.e. if the transformation is \(\mathbf{y} = \mathbf{f}(\mathbf{y})\), then each basis function is a monomial in \(\mathbf{y}\). transform is required to be affine (and non-degenerate) for the basis to span the same space as the polynomials in \(\mathbf{x}\). No checks are performed to verify this.
degreedetermines the degree of the polynomial space.
