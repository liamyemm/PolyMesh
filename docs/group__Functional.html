class  PolyMesh2D::Functional::Family< BasisType >
 A class representing a family of functions expressed in a given basis with coefficients stored in a matrix. This class allows the evaluation of functions in the family and their derivatives at any given point in the domain of the functions. The family is represented by a BasisType object which defines the basis in which the family is expressed, and a coefficient matrix whose i-th line contains the coefficient of the expansion of the i-th function of the family in the basis. Given two functions \(g:\mathbb{R}^n\to\mathbb{R}^k\) and \(f:\mathbb{R}^k\to\mathbb{R}^m\), compose constructs a Function <n, m> for the composition \(f\circ g:\mathbb{R}^n\to\mathbb{R}^m\).

Template Parameters
input_dim( \(n\) in description above) - The input dimension of the first function, \(g\). Must be a strictly positive integer value.
intermediate_dim( \(k\) in description above) - The output dimension of the first function, \(g\), and input dimension of the second function, \(f\). Must be a strictly positive integer value.
output_dim( \(m\) in description above) - The output dimension of the second function, \(f\). Must be a strictly positive integer value.

Generates a scalar-valued shifted monomial basis in the parameter of a Curve. i.e. If the Curve is defined by \(E=\{\gamma(t)\}\) then the basis is in the coordinate \(t\).

Parameters

Generates a basis for the Raviart-Thomas-Nedelec space \(\mathbb{RTN}^{k}(T) = \mathbb{P}^{k-1}(T)^d + \mathbf{x}\mathbb{P}^{k-1}(T)\) in 2 dimensions.

Parameters
transformdescribes a coordinate transformation under which the basis is described. i.e. if the transformation is \(\mathbf{y} = \mathbf{f}(\mathbf{x})\), then the basis for the the Raviart-Thomas-Nedelec is given by vector valued monomials in \(\mathbf{y}\) plus \(\mathbf{y}\) times scalar valued monomials in \(\mathbf{y}\). transform is required to be affine (and non-degenerate) for the basis to span the same space as the Raviart-Thomas-Nedelec space in \(\mathbf{x}\). No checks are performed to verify this.
degreedetermines the degree of the Raviart-Thomas-Nedelec space.
