class  PolyMesh2D::Functional::Family< BasisType >
 A class representing a family of functions expressed in a given basis with coefficients stored in a matrix. This class allows the evaluation of functions in the family and their derivatives at any given point in the domain of the functions. The family is represented by a BasisType object which defines the basis in which the family is expressed, and a coefficient matrix whose i-th line contains the coefficient of the expansion of the i-th function of the family in the basis. Extracts a component from a higher-dimensional function.

This method takes a higher-dimensional function and extracts a single component from it, resulting in a new function with a reduced output dimension of 1. The component to extract is specified by the index parameter.

Template Parameters
input_dimThe dimension of the input space.
output_dimThe dimension of the output space of the original function.
Parameters
funcThe original function from which to extract the component.
indexThe index of the component to extract.
Returns
A new function with a reduced output dimension of 1, representing the extracted component.
Precondition
index must be less than output_dim of the original function.

const VectorBasis2D PolyMesh2D::Functional::RaviartThomasNedelecBasis ( const VectorFunction2D &  transform,
const size_t  degree 
)

Generates a basis for the Raviart-Thomas-Nedelec space \(\mathbb{RTN}^{k}(T) = \mathbb{P}^{k-1}(T)^d + \mathbf{x}\mathbb{P}^{k-1}(T)\) in 2 dimensions.

Parameters
transformdescribes a coordinate transformation under which the basis is described. i.e. if the transformation is \(\mathbf{y} = \mathbf{f}(\mathbf{x})\), then the basis for the the Raviart-Thomas-Nedelec is given by vector valued monomials in \(\mathbf{y}\) plus \(\mathbf{y}\) times scalar valued monomials in \(\mathbf{y}\). transform is required to be affine (and non-degenerate) for the basis to span the same space as the Raviart-Thomas-Nedelec space in \(\mathbf{x}\). No checks are performed to verify this.
degreedetermines the degree of the Raviart-Thomas-Nedelec space.
