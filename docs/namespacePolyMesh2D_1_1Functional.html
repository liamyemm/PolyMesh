class  Family
 A class representing a family of functions expressed in a given basis with coefficients stored in a matrix. This class allows the evaluation of functions in the family and their derivatives at any given point in the domain of the functions. The family is represented by a BasisType object which defines the basis in which the family is expressed, and a coefficient matrix whose i-th line contains the coefficient of the expansion of the i-th function of the family in the basis.
