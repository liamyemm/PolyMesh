Detailed Description

template<unsigned input_dim, unsigned output_dim>
class PolyMesh2D::Functional::Function< input_dim, output_dim >

The Function class is a wrapper class which stores a std::function describing the function value and another std::function describing the function derivative.

It describes in an abstract sense a function that maps elements from \(\mathbb{R}^n \to \mathbb{R}^m\), for arbitrary postive integers \(n, m\).

It is the base class for each of ScalarFunction1D, ScalarFunction2D, VectorFunction1D, VectorFunction2D, as well as Curve. It is the core object in the Functional module.
