The Poly\+Mesh library is a C++ library for implementing polytopal methods for partial differential equations, with a particular focus on hybrid high-\/order methods. It allows for the use of very general and possibly curved meshes and for enriched polynomial spaces.

For more information on using the Poly\+Mesh library, see the documentation at \href{https://liamyemm.github.io/PolyMesh/}{\texttt{ https\+://liamyemm.\+github.\+io/\+Poly\+Mesh/}}.

\doxysection*{Dependencies}

The Poly\+Mesh library depends on the following libraries\+:


\begin{DoxyItemize}
\item Eigen library
\item Boost library
\item CMake
\end{DoxyItemize}
