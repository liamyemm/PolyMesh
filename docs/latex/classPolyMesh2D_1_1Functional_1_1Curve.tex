\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classPolyMesh2D_1_1Functional_1_1Curve}{Curve}} is a function which maps values from $[t_0,t_1]$ to $R^2$. It is a sub-\/class of Vector\+Function1D and stores the extra functions tangent, normal, and derivative\+\_\+norm. \mbox{\hyperlink{classPolyMesh2D_1_1Functional_1_1Curve}{Curve}} also stores parameters tmin and tmax which determine the end points of the curve.

\mbox{\hyperlink{classPolyMesh2D_1_1Functional_1_1Function}{Function}} is a templated class describing an abstract function that maps elements from \$R$^\wedge$n\$ to \$R$^\wedge$m\$. It takes in two template parameters -\/ the input dimension and output dimension. \mbox{\hyperlink{classPolyMesh2D_1_1Functional_1_1Function}{Function}} is a wrapper class which stores a std\+::function describing the function value and another std\+::function describing the function derivative.
